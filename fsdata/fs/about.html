About

You are running a demo program on the Raspberry Pi Pico W that uses the Pico SDK and included LwIP library to run a webserver. The program used the Pico_w_connection_manager C++ class to establish a connection to the AP and then started a the LwIP HTTP server app that served this website.

To enable debugging compile with flags -DLWIP_DEBUG=1 -DHTTPD_DEBUG=LWIP_DBG_ON.

For more info see HTTP server documentation.

The overall web page design is based on HTTP server with ESP8266 WebSockets.
